# 📋 ListsJAVA

Este repositório foi criado com o objetivo de praticar e consolidar conceitos de **listas em Java (`List` / `ArrayList`)** juntamente com **encapsulamento**, aplicando boas práticas de programação orientada a objetos.

---

## 🚀 O que foi aprendido

Durante o desenvolvimento deste projeto, foram aplicados na prática os seguintes conceitos:

### 📚 Listas (`List` / `ArrayList`)

* Criação e uso de `List`
* Diferença entre arrays e listas
* Métodos principais:

  * `add()`
  * `remove()`
  * `get()`
  * `size()`
* Iteração com `for` e `for-each`

---

### 🔒 Encapsulamento (POO)

* Uso de atributos `private`
* Criação de **getters e setters**
* Proteção e controle de acesso aos dados
* Organização do código em classes

---

### 🧩 Programação Orientada a Objetos

* Criação de classes e objetos
* Construtores
* Separação de responsabilidades
* Manipulação de listas com objetos

---

### 🔄 Manipulação de Dados

* Inserção e remoção de elementos
* Busca dentro de listas
* Filtragem de dados
* Atualização de informações via métodos

---

## 💻 Tecnologias utilizadas

* Java
* API `List` (`ArrayList`)

---

## 📌 Objetivo do projeto

Este projeto tem como foco o aprendizado prático de **listas dinâmicas** e **encapsulamento**, fundamentais para o desenvolvimento de sistemas mais organizados, seguros e escaláveis.

---

## 📚 Aprendizados importantes

* Diferença entre **array (tamanho fixo)** e **lista (dinâmica)**
* Importância do encapsulamento para proteger dados
* Como trabalhar com objetos dentro de listas
* Base para sistemas reais (cadastro de usuários, produtos, etc.)
